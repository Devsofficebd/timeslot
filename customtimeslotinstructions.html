Our Custom Made Plugin

Using this plugin you can show when you are available without using any premium booking plugin. For example you want to show you are available in just 4 times and each time your capacity to allow maximum 5 students.

You have latest Version Already

STEP 1

Install Woocommerce plugin if you don't have.

STEP 2

Install Custom Time Slot Plugin and activate it.

STEP 3

Create a product using woocommerce and you will see the repeater field option. Write for example [10:00 am - 11:00 am] and capacity for example 5. Publish or update the product then view the product. You will see a dropdown up to add to cart button

Different Way

If you want to show the select dropdown in different page then copy the product ID

FINISH

Edit the page and paste this shortcode with your product id. [time_slot_add_to_cart product_id='paste_your_product_id']
